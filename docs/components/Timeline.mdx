# Timeline

The timeline displays a list of events in chronological order.

# Props

<table>
  <tr>
    <th>Props</th>
    <th>Type</th>
    <th>Required</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>data</td>
    <td>Array`<TimelineData>`</td>
    <td>Yes</td>
    <td>It is an array of objects of `TimelineData` type</td>
  </tr>
  <tr>
    <td>UserTimelineComponent</td>
    <td>React.FC`<unknown>`</td>
    <td>Yes</td>
    <td>The component that will be rendered to create the timeline</td>
  </tr>
  <tr>
    <td>userMilestoneIcon</td>
    <td>string | null</td>
    <td>No</td>
    <td>
      The url of the icon that you want to show for every milestone on the timeline. By default, the `Timeline` component picks the `milestoneIcon` passed in the `data` array prop
    </td>
  </tr>
   <tr>
    <td>milestoneIconSize</td>
    <td>number</td>
    <td>No</td>
    <td>
      The size of the milestone icon. This size will also apply to the image that you can pass to replace the milestone icon. Its default value is `40px`
    </td>
  </tr> <tr>
    <td>showSameMilestoneIcon</td>
    <td>boolean</td>
    <td>No</td>
    <td>
      When `true`, it will show a user provided icon i.e. `userMilestoneIcon` for every milestone, its value is `false` by default
    </td>
  </tr> <tr>
    <td>timelineGap</td>
    <td>number</td>
    <td>No</td>
    <td>
      The gap (margin) between two timeline entries on the page. Its default value is `16px`
    </td>
  </tr>
</table>

# TimelineData

Each object in the `data` array will be used to render a timeline entry. Therefore, every object in the `data` array should be of the following type.

<table>
  <tr>
    <th>Props</th>
    <th>Type</th>
    <th>Required</th>
    <th>Description</th>
  </tr>
  <tr>
    <td>milestoneIcon</td>
    <td>string</td>
    <td>Yes</td>
    <td>
      The url of the the icon which you want to show for that particular
      milestone
    </td>
  </tr>
  <tr>
    <td>componentPropsData</td>
    <td>`object`</td>
    <td>Yes</td>
    <td>
      This will be used as the prop object for `UserTimelineComponent`. The type
      `object` literally means `any object`, hence it can be passed as the prop
      object to the `UserTimelineComponent` which accepts the prop object of
      `unknown` type i.e. `
      <UserTimelineComponent {...componentPropsData} />`
    </td>
  </tr>
</table>
