# Canary Design

Canary Design is a UI component library built in React. Its built using Vite's library mode and supports TypeScript out of the box.

## Getting Started

Install canary-design using npm:

<pre>npm i canary-design</pre>

Now that you are done with the installation, lets check out the [components](/docs/category/components).
